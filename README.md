## Main features

### Common
- Recruiter role - Company:
  - Create recruitment news, update jobs
  - Check candidates'CV by job
  - Update company's info
- Seeker role:
  - View recruitment news, detail recruitment, related jobs.
  - Search job with filter
  - Apply every job in just a few steps
  - Save your favorite jobs
  - Manage history of jobs application

### Special:
- Allows seekers to message the recruiters and vice versa. This helps the seekers approach to jobs or recruiters quickly.
